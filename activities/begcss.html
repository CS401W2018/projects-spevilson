Questions

1.Explain the difference between inline, embedded, and external CSS.

Inline CSS is putting in attributes using the style attribute within each tag. Embedded CSS is using the style tag in the head section. External CSS is having a separate file that dictates the style of the web page.

2.Explain why you should only use external CSS.

External CSS is the most widely used due to the ease of editing, and due to the same CSS file being able to apply to many HTML files.

3. What is a selector used for?

A selector, represented by curly braces, is used in CSS to apply attributes to tags.
