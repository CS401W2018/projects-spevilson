Questions

1. What should you focus on during class and after class?

In class, we should be following along and answering questions, and after after class we should be finishing assignments and making sure that we understand the concepts.

2. What is the main difference between CSS Grid and CSS Flex?

Grid is a two dimensional system, while flex is one dimensional. Flex can also be used for two dimensional, but we will learn more about that later. Grid is an authoritarian parent, and flex is flexible.
