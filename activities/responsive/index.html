How can you adjust the CSS for printing?

You can add the 'link' tag and link a separate css file, and then select the "print" media instead of the "screen" media.

When would you want to adjust the CSS for printing?

You would want to adjust the CSS for printing when you have a lot of words or when you expect people to want to print your webpage.
