How can you adjust the CSS for printing?

You can add the 'link' tag and link a separate css file, and then select the "print" media instead of the "screen" media.

When would you want to adjust the CSS for printing?

You would want to adjust the CSS for printing when you have a lot of words or when you expect people to want to print your webpage.

Questions 3/5

Why do we need a viewport meta tag?

The viewport meta tag tells the computer not to zoom the webpage and mess with it when it is opened on a mobile phone.

When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

The tag that comes after the other one in the CSS takes priority.

How do you change the CSS for a web page displayed on a phone?

to change the CSS for a webpage on a phone, you add an exception at the end that only happens when certain conditions are met.
