Questions

What is JavaScript? How did it get its name?

JavaScript was created in order to be run in a browser. It got its name when Sun MicroSystems named it after the Java programming language.

What are the two ways you can run JavaScript?

The first way is in the DOM, or in the browser. The second way is through node.js, which developers created in order to use JavaScript with servers.

What was the most popular front-end js library?

It used to be jQuery. It became really popular because it would make it so JavaScript was usable on any browser without edits.

What is the most popular front-end js library today?

One of the most popular, if not the most popular, is React.js.

What is AJAX?

AJAX is a way for webpages to send and receive data asynchronously, which means that there's no need to reload the webpage every time.

Why do you put your JS Code in a separate file?

Itis similar to putting your CSS in a separate file. It is cleaner and easier to use, and it is easier for people that only are experts in one programming language instead of both.
