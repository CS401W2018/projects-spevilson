What is the difference between block tags and inline tags?

Block tags create blocks in the page that are separated. They are all basically div tags. Inline tags, however, are within a line of text and don't create a new box.

What is the margin, border, and padding?

The margin is the area around the tag before the other tags are reached. Border is a box around the tag, and padding is an extended version of the tag around the tag.

What are units of measurement you can use in CSS?

You can use centimeters, inches, pixels, and pts for font.

When would you use a figure tag?

You would use a figure tag when you want an inline tag to act as a block tag.
