4. List the 3 rules with relative links.

The three rules pertain to parent-to-nephew, child-to-parent, and sibling-to-sibling, with any combination of those. When we go from parent to nephew, we just print the name of the folder the child is in, plus the name of the child. For example:

<a href="folder1/second.html">second</a>

Another example is when we need to go up the tree. Then we use "../".
