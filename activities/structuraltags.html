Structural Tags

(HEADER, NAV, MAIN, FOOTER)

1. What goes in the header?

The header is where we place the logo and sometimes tha links for navigation throughout the page, as a map or a home-base of some kind.

2. What goes in the Nav? Can the Nav go in the header? Does it have to go in the header? Does the Nav contain links to another website?

The Nav is for links that help the user navigate through the website. The Nav can go in the header, but doesn't need to. I think that typically the Nav does go inside, but that is optional. The Nav only contains links to other parts of the website, kind of like the beginning of a tree.

3. What goes in Main?

Main is for the main body of the website. The body tag is everything on the page, and the main tag is for everything that is typed out, that isn't a hyperlink or something like that.

4. What goes in the Footer?

The footer normally contains copyrights, contact information, and links to help or FAQ pages.

5. What is HTML Validation?

HTML validation, whether through VSCode extensions or through websites, is a way to make sure that the HTML you're using is correct.
