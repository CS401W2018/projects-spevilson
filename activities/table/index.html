Questions

When do you use a table?

You would use a table whenever you would want to use Excel. Any data that you would want to display you would use a table.

When do you not use a table?

You wouldn't use a table in situations like a side column or where a grid display would be more appropriate. A lot of data, or extra big datum, would make it a lot harder to use a table.

When do you need to do extra work for table accessibility?

You need to do extra work for table accessibility every time. If you want customers, especially customers that will come back, you should make your page accessible.
