Internet

The internet is a big network of computers and devices that talk and work together. It's the way that we communicate and access information.

Web

The Web is like a big library with really fast robot librarians that go and get the information that we need or ask for.

How are the Internet and Web different?

If the Web is the system of librarians and books that make up the library system and make it effective, the Internet is the building, the tracks the robot librarians are on, and the way that information gets everywhere.

ASCII and Unicode

ASCII and Unicode are ways to numerically represent letters, accents, and typing characters. ASCII is limited to the English language and roman characters, while Unicode is internationally compatible.

What is UTF-8?

UTF-8 stands fr Unicode Transformation Format - 8 bit. It turns binary numbers in bytes (each formed by 8 bits)into letters, based upon Unicode and ASCII codes. Example: A = 01000001

URL

Staying with the library metaphor, a URL (or Uniform Resource Locator) is the location of the book in the library that the user sees, and normally includes some sort of title for the page.

Domain Name

The domain name is the title of the book in the library, easily searched by the person looking for the book, as opposed to the dewey decimal system, which would be harder.

HTTP

HTTP, or Hypertext Transfer Protocol, is the way that the person searching is connected to the server that holds the information that they need.

HTTPS

HTTP Secure, or HTTPS, is an encrypted version of HTTP.

IP Address

IP Address, or Internet Protocol Address, is the numerical address of each device or page on the Web. It is 4 digits that can each be represented by a bit, so all the way up to 255. For example, Google's IP address is 8.8.8.8

DNS

The DNS, or Domain Name System, is the way that IP Addresses are matched, corresponded, and connected to URLs.

W3C Recommendations

WWWC Recommendations are standards that each participant in the Web must follow to be a part of it.

XML, XHTML, HTML, HTML5

XML, or eXtensible Markup Language, is a language for creating documents that has tags that can be added to or extended. This is different than HTML, which is similar, but unable to be extended. HTML5 is the current version of HTML, and XHTML is like a mixture of XML and HTML.
